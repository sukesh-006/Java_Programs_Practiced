## 📖 About This Repository
Welcome to my Java Practice Repository! This workspace is a curated collection of Java programs I have written to solidify my understanding of object-oriented programming (OOP), data structures, algorithms, and logical problem-solving. 

I strongly believe that **consistency is the key to mastery**. This repository serves as a testament to my dedication to continuous learning, clean coding practices, and taking on progressively harder challenges. 

Every single file in this repository is thoroughly documented, appropriately named, and properly structured to ensure maximum readability and maintainability.

## 🚀 Key Topics Covered
Through the programs in this repository, I have implemented and explored:

*   **Core Logic Building**: Number theory, basic math operations, and digit manipulations.
*   **Control Flow**: Deep dives into `if-else` conditionals, `switch` statements, and complex `while`/`for` loops.
*   **Pattern Printing**: Pyramids, diamonds, and inverted structures to master nested loops.
*   **Arrays & Strings**: 1D/2D arrays, searching algorithms (Linear & Binary Search), anagrams, and palindromes.
*   **Methods & Recursion**: Function overriding, mathematical recursions (Factorial, Fibonacci, GCD), and modular programming.
*   **Collections Framework**: Hands-on implementations using `ArrayList`, `LinkedList`, `Stack`, `Queue`, `HashSet`, `LinkedHashSet`, and `TreeMap`.
*   **Problem Solving**: Real-world competitive programming challenges (e.g., Toll Calculations, Delivery Discounts, Pair Sums).

## 🛠️ How to Run Locally
To run any of the programs locally, ensure you have the [Java Development Kit (JDK)](https://www.oracle.com/java/technologies/downloads/) installed on your machine.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/your-repo-name.git
   cd your-repo-name
   ```
2. **Compile the Java file:**
   Navigate to the directory of the file and compile it:
   ```bash
   javac FileName.java
   ```
3. **Execute the compiled program:**
   ```bash
   java FileName
   ```
